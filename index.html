httpirc by abh

HTTP to IRC gateway

httpirc allows you to post messages to IRC servers via HTTP. It's useful to have an easy way for your monitoring system, bug tracker etc post messages to your company or project IRC server / channel.

Dependencies

perl and the following perl modules: AnyEvent, AnyEvent::IRC, JSON

Install

cpan AnyEvent
cpan AnyEvent::IRC
cpan JSON

./httpirc.pl --irc-server irc.example.com --irc-port=6667 --irc-nick http --http-port 9090
